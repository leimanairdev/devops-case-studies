# DevOps Case Studies

A portfolio-ready collection of DevOps case studies demonstrating:
- Infrastructure as Code (Terraform)
- CI/CD automation (GitHub Actions)
- Incident response and observability practices

## Projects

### AWS Network Baseline (Terraform)
Secure multi-AZ VPC foundation with public/private subnets, routing, and NAT for private egress.
- Folder: `aws-network-baseline-terraform/`

### CI/CD to Kubernetes (GitHub Actions)
Build/test pipeline + deployment strategy for a containerized application targeting Kubernetes.
- Folder: `k8s-cicd-github-actions/`

### Incident Response: Latency & 504 Errors
Postmortem-style writeup covering timeline, investigation approach, monitoring gaps, and prioritized remediation.
- Folder: `incident-response-postmortem/`
